Some additional compute services?
  1. Elastic Beanstalk
  2. AWS Batch
  3. Lightsail
  4. Outposts

    - Elastic Beanstalk
        it's a fully managed service that deploy your application,Restful Apis,backend...
        you don't need to manage all the underlinning resources
        it handls all the infrastructure,load balancing,scaling,application health monitoring
        supports varies programming langugages
        automating many operational tasks.
    -AWS Batch 
       - fully managed service for batch computing service
       - good for services that needs; large-scale,parallel workloads (scientific research,ML,genomics research,financial analysis ,high performance computations...
    - Lightsail
        - service offers ; VPS(virtual pervite servers),databases,storage,networking at low level.
        good for ;  small developing,learning cloud services,blogs,small apps

    - Outposts;
        - service 
            hybrid  on-premises cloud -> provide cloud for local servers on a remote company
            this offers storage,networking,...
        good for: low-latency apps,data processing in  remote places...
